# Contributing

Thanks for contributing to AI/ML Core Notes.

## Principles

- Keep content practical and beginner-friendly.
- Explain concepts in simple language first, then add depth.
- Include references to reliable sources.
- Prefer examples that can be adapted to real projects.

## Content format

- Place topic docs under `docs/` in the correct section.
- Add a short summary first, then deeper explanation.
- Add at least one practical example.
- Include links to packages/tools in Python, Julia, and R when relevant.

## Pull requests

- Keep PRs focused and small.
- Add or update links in section indexes.
- Ensure checks pass.
