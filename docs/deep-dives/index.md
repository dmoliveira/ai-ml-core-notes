# Deep Dives

Use this section when you want full depth.

## When to use deep dives

Choose deep dives after you can comfortably complete the foundations checklist and want stronger reasoning for interviews or system design.

Recommended loop:

1. Pick one weak topic from `foundations/index.md`.
2. Study one deep-dive thread end-to-end.
3. Capture a one-paragraph takeaway and one trade-off you learned.
4. Validate understanding with a related prompt from `interview-prep/question-bank.md`.

- Expanded theory notes
- Architecture walkthroughs
- Benchmarking and evaluation protocols
- Real-world case studies

## Output you should produce

- A concise decision note (what you would choose and why).
- One metric plan for validation.
- One failure mode and mitigation strategy.
