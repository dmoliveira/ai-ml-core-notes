# Summaries

Use this section for rapid revision.

## When to use summaries

- Before practice sessions to reactivate key concepts quickly.
- Before interviews to refresh frameworks and trade-off language.
- After deep study to retain only high-signal takeaways.

- One-page concept maps
- Interview quick notes
- Decision cheat sheets
- Common mistakes and fixes

## 15-minute revision loop

1. Review one summary page.
2. Explain the topic out loud in your own words.
3. Answer one question from `interview-prep/question-bank.md`.
4. Run one related exercise from `practice/index.md`.

## Progress signals

- You can explain a concept without rereading the page.
- You can name one practical trade-off and one failure risk.
- Your practice and interview answers become shorter and clearer.
