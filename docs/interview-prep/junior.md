# Junior Interview Track

Use this track if you are preparing for internship or junior AI/ML roles.

## What interviewers look for

- Clear understanding of supervised vs unsupervised learning
- Correct use of train/validation/test splits
- Basic model evaluation using precision, recall, F1, and ROC-AUC
- Ability to explain overfitting and regularization

## Study checklist

1. Explain bias-variance trade-off in plain language
2. Compare logistic regression and tree-based models
3. Interpret confusion matrix and threshold changes
4. Build one tabular model end-to-end on Kaggle

## Mini practice

- Dataset: Titanic or House Prices on Kaggle
- Goal: submit one baseline and one improved model
- Deliverable: write a one-page model decision note
